# Inventory & Billing Management (Streamlit)

This repository contains a simple Inventory and Billing Management system built using Python and Streamlit.

## Files included
- app.py – Main Streamlit app  
- billing.py – Billing logic  
- database.sql – Database schema  
- requirements.txt – Project dependencies  
- ss add product.png – Add product page screenshot  
- ss billing.png – Billing page screenshot  
- ss sales summary.png – Sales summary page screenshot  

## Features
- Add products  
- Generate bills  
- View daily sales summary  

This project is for learning and practice purposes.
